## Setup environment
Steps to setup environment in azure and local machine:

1. Setup Azure Databricks
2. Setup ADLS2
    - Create storage account
    - Create container name `ecommerce`, with three folder `bronze`, `silver`, `gold`
    - Upload data
3. Setup Airflow
4. Setup dbt
5. Setup Docker
6. Setup Git
7. Setup Python
